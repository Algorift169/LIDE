⚠️  TROUBLESHOOTING:

Q: Files don't open with File Roller
A: Run: ./fileRoller/setup-mime-associations.sh

Q: File Roller not in "Open With" menu
A: Update desktop database:
   update-desktop-database ~/.local/share/applications/

Q: Wrong application opens
A: Run setup script again:
   ./fileRoller/setup-mime-associations.sh
